# Installation
Clone the module into your MagicMirror module folder and execute `npm intall` in the module's directory.

 ```sh
 cd ~/MagicMirror/modules
 git clone https://github.com/bugsounet/EXT-Updates
 cd EXT-Updates
 npm install
 ```
