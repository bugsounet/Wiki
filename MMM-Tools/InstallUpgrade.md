Install and Upgrade commands

# New Install

```sh
cd ~/MagicMirror/modules
git clone https://github.com/bugsounet/MMM-Tools
cd MMM-Tools
npm install
```

# Upgrade

```sh
cd ~/MagicMirror/modules/MMM-Tools
git pull
npm install
```
