# Require

* MagicMirror v2.18.0 and more
* `EXT-Browser` for open news in fullscreen (optional)

# Installation

In a terminal, try this commands:

```sh
cd ~/MagicMirror/modules
git clone https://github.com/bugsounet/MMM-NewsFeed
cd MMM-NewsFeed
npm install
```
